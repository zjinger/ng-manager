# 使用指南

这部分文档面向已经能启动 `ng-manager` 的用户，重点说明日常开发中最常用的能力。

## 你会在这里看到什么

- [项目管理](/guides/projects)：如何导入、识别和维护本地项目。
- [任务执行](/guides/tasks)：如何运行脚本、查看状态、处理失败任务。
- [日志与排障](/guides/logs)：如何定位问题、观察输出、缩小故障范围。
- [CLI](/guides/cli)：如何用 `ngm` 命令启动、重启、查看状态。
- [仪表盘](/guides/dashboard)：如何使用可定制部件搭建首页工作台。
- [依赖管理](/guides/dependencies)：如何查看、安装、更新和卸载依赖。
- [配置编辑](/guides/configuration)：如何用结构化表单管理配置文件。
- [API 调试](/guides/api-client)：如何管理请求集合、环境变量和历史记录。
- [雪碧图](/guides/sprite)：如何同步资源、生成雪碧图并查看结果。

## 推荐阅读顺序

1. 先看 [项目管理](/guides/projects)，把工程正确导入进来。
2. 再看 [任务执行](/guides/tasks)，建立日常操作路径。
3. 最后看 [日志与排障](/guides/logs) 和 [CLI](/guides/cli)，提升排查效率。

## 按角色快速定位

- 日常开发者：优先阅读 [仪表盘](/guides/dashboard)、[任务执行](/guides/tasks)、[日志与排障](/guides/logs)。
- 项目维护者：优先阅读 [依赖管理](/guides/dependencies)、[配置编辑](/guides/configuration)。
- 工具链使用者：优先阅读 [API 调试](/guides/api-client)、[雪碧图](/guides/sprite)、[CLI](/guides/cli)。
