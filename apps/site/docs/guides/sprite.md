# 雪碧图

雪碧图页面面向资源整合场景，重点处理图标资源同步、生成结果预览与日志查看。

## 页面能力

- 配置雪碧图参数。
- 从 SVN 同步资源。
- 生成雪碧图。
- 查看最近同步时间。
- 展示生成结果。
- 打开侧边日志面板查看执行输出。

## 典型工作流

1. 首次进入时，先点击设置完成雪碧图配置。
2. 如果资源来自 SVN，先执行同步。
3. 打开日志面板观察同步或生成过程。
4. 执行“生成雪碧图”。
5. 在结果区域确认输出是否符合预期。

## 适合的使用场景

- 团队仍在维护基于图标位图的前端资源体系。
- 需要把资源同步与生成流程从手写脚本中抽出来。
- 需要在生成后立即比对输出结果和日志。

## 使用建议

- 首次配置完成后，先做一次完整同步再生成，避免增量状态不一致。
- 出现结果异常时，先看右侧日志，再检查资源路径和源配置。
- 如果项目的图标资源来自多人维护，建议把配置规则也纳入版本管理。
