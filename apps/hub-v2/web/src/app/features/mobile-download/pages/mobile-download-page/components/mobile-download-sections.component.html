安装指南

iOS 安装

  1. 点击上方「下载安装包」或扫描二维码
  2. 弹出安装提示时点击「安装」
  3. 打开 设置 - 通用 - VPN与设备管理
  4. 找到企业证书，点击「信任」
  5. 返回桌面打开 Hub V2 Mobile

Android 安装

  1. 点击上方「下载安装包」或扫描二维码
  2. 打开下载的 .apk 文件
  3. 如提示「未知来源」，在设置中允许安装
  4. 按提示完成安装
  5. 打开 Hub V2 Mobile 并配置服务器地址
