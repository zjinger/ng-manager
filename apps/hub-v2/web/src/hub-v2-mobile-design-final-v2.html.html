Hub 是一个企业协作与管理平台，帮助团队实现研发协同、任务流转、文档共享和项目管理。
1. 我们会遵循隐私政策收集、使用必要的个人信息，不会强制收集与当前功能无关的信息。
2. 通讯录、相册、日历等权限默认不会开启，仅在你使用相关功能并明确授权后调用。
3. 你可以在系统设置中管理或撤回授权，撤回后不会影响其他基础协作功能。
4. 我们会采取安全措施保护你的个人信息，仅在企业协作、身份验证、消息通知、项目管理等必要场景中使用。
