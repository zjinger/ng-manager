提交审批

确认后单据将进入负责人节点。

转交审批

当前节点将由新审批人继续处理，原审批人不再承担本节点审批责任。

适用于审批人请假、流转错误、职责变更等场景。
流转结果：部门主管（当前你） → 李明（代理主管） → 审核 → 会计 → 出纳 → 完成

加签确认

邀请其他人员补充审批意见，完成后回到当前节点继续审批。

适用于费用归属不清、金额较大、需要项目负责人或行政负责人确认等场景。
流转结果：部门主管（当前你） → 赵宇（加签确认） → 部门主管（继续审批） → 审核 → 会计 → 出纳 → 完成
