Excel / XLSX

创建、检查和编辑 Microsoft Excel 工作簿和 XLSX 文件。支持可靠的公式、日期、类型、格式、重新计算和模板保持。适用于数据分析、报表生成、财务建模等场景。

使用场景

  • 数据分析 — 读取、清洗和转换 Excel 数据
  • 报表生成 — 自动化创建格式化的 Excel 报表
  • 财务建模 — 构建带公式的财务模型
  • 数据验证 — 校验数据类型和格式

核心功能

工作簿操作

  • 创建新的 XLSX 工作簿
  • 读取和解析现有 Excel 文件
  • 支持多 Sheet 操作
  • 保持模板格式和样式

公式与计算

  • 支持 Excel 内置函数（SUM、VLOOKUP、IF 等）
  • 自动重新计算
  • 单元格引用和跨 Sheet 引用

格式与样式

  • 字体、颜色、对齐方式
  • 边框和背景填充
  • 列宽行高自适应
  • 条件格式

快速开始

安装后，直接在对话中描述你的需求即可：

# 创建一个包含销售数据的 Excel 文件
请帮我创建一个 Excel 文件，包含以下列：
- 产品名称
- 单价
- 数量
- 总价（公式计算）

# 读取并分析现有文件
请读取 sales-2026.xlsx 并按月份汇总销售额

高级用法

模板保持

使用现有文件作为模板，技能会自动保持原有格式：

请使用 template.xlsx 作为模板，
在 Sheet2 中添加新的数据行

数据验证

支持对单元格设置数据验证规则：

请在 B2:B100 范围内设置数据验证：
- 类型：整数
- 最小值：0
- 最大值：10000
- 错误提示：请输入有效的金额

注意事项

该技能使用纯 Node.js 预编译二进制完成全部操作，无需外部图形库，适合在各类服务器环境中部署。兼容 Excel 2016 及以上版本。
