NGM-042

登录接口偶发 500 错误

待处理 进行中 已解决 已验证 已关闭

在高并发场景下，/api/admin/auth/login 接口偶发返回 500 Internal Server Error。


复现步骤：

1. 使用 JMeter 模拟 50 并发登录请求

2. 持续运行 5 分钟

3. 约 2% 的请求返回 500


期望：所有请求正常返回 200 或 401

实际：偶发 500 错误，日志显示 SQLite database is locked
