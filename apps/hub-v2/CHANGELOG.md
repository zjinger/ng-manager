# Changelog

记录 `apps/hub-v2` 的版本变更（SemVer）。

## [2.1.1] - 2026-04-07

### 主要升级

- 新增问卷调查模块，支持问卷编辑、发布与多页能力。
- 积木报表能力增强：支持多图表块联动渲染、中文化文案优化、模板看板组合与渲染。
- 新增积木报表公开看板发布与管理能力（发布、失效、恢复、删除）。
- 测试追踪（Issue）列表增强：类型多选筛选、批量标记已解决、提报人可编辑。
- 全局搜索能力补强：接入后端搜索服务，完善高亮与交互一致性。
- AI 能力增强：新增 Issue 推荐（分类/优先级/指派）与报表 SQL 生成能力优化。
- 实时能力增强：通知、活动时间线、事件总线与 WebSocket 连接策略优化。
- 登录与安全链路升级：RSA/Challenge 兼容增强、密码与环境变量校验强化。

### 修复

- 修复积木报表归档项目过滤问题，避免归档项目进入公共报表数据范围。
- 修复生产环境 AI 接口配置问题（API Key / Base URL）。
- 修复认证兼容问题：HTTP 场景下 challenge + AES 登录链路可用，并保留 RSA 兼容。
- 修复 Hub Token 读取接口与 projectKey 路由安全性问题。

### 发布说明

- 当前版本升级说明文件：`release-notes/v2.1.1.md`
- 升级建议：先执行 `npm run db:migrate`，再重启服务并回归验证全局搜索、测试追踪、积木报表。
