# Changelog

记录 `apps/hub-v2` 的版本变更（SemVer）。

## [2.1.0] - 2026-04-03

### 主要升级

- 全局搜索能力完善（含命中高亮、分组展示与权限口径修正）。
- AI 测试追踪增强（新增 issue 推荐、模块推荐链路完善）。
- AI 积木报表完成 MVP 并增强多图联动、模板看板与紧凑布局。
- issue 列表支持类型多选筛选、默认创建时间倒序、批量标记已解决。

### 质量修复

- 修复多处报表 SQL 字段与聚合口径问题。
- 修复线上全局搜索接口兼容性异常（500 / 字段不存在场景）。
- 修复项目名称/项目 ID 展示异常与图表重复项目问题。
