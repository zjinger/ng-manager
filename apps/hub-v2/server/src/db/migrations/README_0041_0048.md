# Hub-v2 Migration Index (0041-0048)

This index documents the intent and dependency order for the current admin/RBAC/organization/expense foundation migrations.

## Ordered Scope

1. `0041_user_org.sql`  
   Organization base: `departments`, final-form `user_departments` (single primary department per user).

2. `0042_system_rbac.sql`  
   Unified RBAC base: `system_roles`, `system_permissions`, role bindings, initial seeds.

3. `0043_approval.sql`  
   Approval template and stage metadata (fixed-stage template model).

4. `0044_reimbursement.sql`  
   Reimbursement core tables: claims, items, attachments, tasks, logs.

5. `0045_project_rbac.sql`  
   Project-governance related permission seeds into `system_permissions`.

6. `0046_system_settings.sql`  
   Global system settings storage.

7. `0047_project_titles.sql`  
   Project role dictionary (`project_titles`) with active/inactive status, and user default project title (`users.default_project_title_code`).

8. `0048_organization_titles.sql`  
   Organization title dictionary (`organization_titles`), `users.organization_title_code`, and department-to-organization-title mapping (`department_titles`).

## Notes

- `0047_user_departments_single_primary.sql` and `0049_system_permissions_status.sql` were removed before deployment, and the final schema was folded into the remaining migrations.
- Project roles and organization titles are separated: project roles describe project collaboration duties, while organization titles describe employee title data used by departments and reimbursement snapshots.
