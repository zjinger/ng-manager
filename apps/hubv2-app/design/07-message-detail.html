v2.3 版本发布通知
📅 2025-06-09 14:30 👤 系统管理员

更新内容

Hub V2 v2.3 版本已正式发布，本次更新包含以下重要改进：

新功能

  • 研发项进度可视化看板
  • Issue 批量指派与转移
  • 消息中心支持 Markdown 渲染
  • 移动端底部导航优化

改进

  • 登录接口性能提升 40%
  • WebSocket 连接稳定性增强
  • 文件上传支持断点续传

修复

  • 修复了项目切换后数据未刷新的问题
  • 修复了消息推送延迟的问题
  • 修复了暗色模式下部分文字对比度不足的问题

升级指南

请按照以下步骤完成升级：

# 拉取最新代码
git pull origin main

# 安装依赖
pnpm install

# 执行数据库迁移
pnpm migrate

# 重启服务
pnpm restart

注意：升级前请确保已备份数据库。维护窗口为 6月15日凌晨 2:00-4:00。

已知问题

部分用户在 Safari 浏览器上可能遇到样式兼容问题，我们将在 v2.3.1 中修复。
