Server 名称 域名
@for (domain of commonDomainOptions; track domain) { } @for (port of commonListenOptions; track port) { }
域名和端口均支持多个值，例如：localhost, 127.0.0.1 / 80, 443
 根目录
@for (item of commonIndexOptions; track item) { }
根目录示例：D:\dist\www；默认首页示例：index.html, index.htm
 Location 规则
@for (loc of formData.locations; track $index) {
Location {{ $index + 1 }}
}
常见场景：`location /api/` + `proxy_pass http://127.0.0.1:6808;`
 @if (formData.protocol === 'https') { SSL 证书路径 SSL 私钥路径 } 自定义配置片段
用于补充高级指令（`location` 常规场景建议在上面的 Location 规则中配置）
